Backend API - RESTful Service

Backend Project • 2024-2025

Project Overview

A scalable RESTful API backend built with Node.js and Express.js, featuring MongoDB database integration and secure JWT authentication. This API serves as the foundation for various web and mobile applications, providing robust data management capabilities.

Key Features:

  • User authentication and authorization with JWT tokens
  • CRUD operations for multiple data models
  • Input validation and error handling middleware
  • MongoDB database with Mongoose ODM
  • Environment configuration for different deployment stages
  • Rate limiting and security headers
  • Comprehensive API documentation
Node.js Express.js MongoDB Mongoose JWT REST API Postman Git

API Endpoints

GET
Get All Users
/api/users

Retrieve a list of all registered users (admin only)

Register User

Installation & Setup

# Clone the repository git clone https://github.com/Tiisetso012/backend-api.git # Navigate to project directory cd backend-api # Install dependencies npm install # Set up environment variables cp .env.example .env # Edit .env with your configuration # Start development server npm run dev # Start production server npm start
